Admin Portal: Management Capabilities

What should the global admin be able to do?

A

User Management

View all users, manually change tiers (Free/Pro), and disable accounts.

B

System-wide Task Monitoring

View all scheduled tasks across the system, identify failing ones, and see global failure rates.

C

Direct DB Access / Shell

A web-based interface to run queries against the tasks and logs tables.

D

Dynamic Quota Configuration

Change 'Free' vs 'Pro' limits (e.g., from 5 to 10 tasks) without restarting the server.
