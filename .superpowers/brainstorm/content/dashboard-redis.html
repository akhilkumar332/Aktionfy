Dashboard Live Updates via Redis

Enhancing the User and Admin dashboards with real-time capabilities using Redis Pub/Sub.

User Dashboard

Redis Pub/Sub + SSE
  • Live Task Status: See tasks transition from `scheduled` → `running` → `completed`/`error` instantly.
  • Live Log Streaming: Watch the output of scheduled actions (e.g., Gemini's response or command output) stream in real-time.
  • Action Required Alerts: Instant pop-ups if a task requires human approval (the `require_plan_approval` feature).

Admin Dashboard

Redis Metrics + Pub/Sub
  • Live System Health: Real-time graphs showing active workers, queue depth, and memory usage.
  • Global Task Feed: A live ticker of all tasks executing across the system (anonymized if necessary).
  • Rate Limit Monitoring: View users hitting API limits in real-time, powered by Redis-based rate limiters.
